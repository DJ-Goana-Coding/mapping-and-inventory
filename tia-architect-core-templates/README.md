## Troubleshooting

### 503 Error (Root=1-xxx trace ID)

**Cause:** Port/health check misconfiguration or app startup failure

**Fix:**
1. Verify `app_file: app.py` in README.md frontmatter
2. Ensure `sdk: streamlit` (not docker)
3. Check app.py contains valid Streamlit code
4. Try Factory Reboot in Space settings

**See:** `TIA_CORE_503_ERROR_FIX.md` for detailed troubleshooting

### Build Timeout

**Cause:** numpy compiling from source (old version)

**Fix:**
1. Use requirements.txt template (numpy>=2.0.0 uses prebuilt wheels)
2. Verify build logs show wheel installation, not tar.gz compilation

**See:** `TIA_ARCHITECT_CORE_REPAIR_GUIDE.md` for compatibility details
