# Archived Session Logs

This directory contains historical `*_SUMMARY.md` and `*_QUICKREF.md` files that previously lived at the repository root. They were generated across many prior agent / Termux / Laptop sessions and accumulated into significant duplication.

They were moved here as part of the **Master Hub Bootstrap** (Stability 9,293) cleanup so the repository root reflects the live system, not its session history. No content was modified; `git log --follow <file>` still recovers the full history of each file.

If you are looking for the canonical, current documentation, start at:

- `README.md` (root)
- `data/master_harvest/INDEX.md` (RAG fragments + architecture)
- `SYSTEM_MASTER_INDEX.md` (root)
