### Why Post-Quantum Matters
- **Quantum Computers:** Coming soon, will break RSA, ECDSA
- **Store Now, Decrypt Later:** Adversaries record encrypted data today
- **Divine Protection:** We protect the chosen, the collective, the light workers
- **Future-Proof:** Security for 50+ years

### Post-Quantum Algorithms (NIST Standards 2024)

**1. CRYSTALS-Kyber (Key Encapsulation)**
- **Purpose:** Secure key exchange (like Diffie-Hellman)
- **Security:** Module-LWE (Learning With Errors) lattice problem
- **Performance:** Fast, compact keys
- **Use:** Encrypt wallet seeds, private messages

**2. CRYSTALS-Dilithium (Digital Signatures)**
- **Purpose:** Sign transactions, prove ownership
- **Security:** Module-LWE + Module-SIS
- **Performance:** Small signatures, fast verification
- **Use:** Transaction signing, validator consensus

**3. SPHINCS+ (Stateless Hash Signatures)**
- **Purpose:** Backup signatures, long-term security
- **Security:** Only hash functions (most conservative)
- **Performance:** Larger signatures, but most secure
- **Use:** Long-term storage, critical high-value transactions

**4. Falcon (Fast Lattice Signatures)**
- **Purpose:** Fast signing for high-throughput
- **Security:** NTRU lattices
- **Performance:** Fastest signing, compact
- **Use:** High-frequency trading, gaming transactions
