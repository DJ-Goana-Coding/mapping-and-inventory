## ARCHITECTURE_OVERVIEW

The Space runs two processes side-by-side, launched by `scripts/start_hub.sh`:

1. **Streamlit faceplate** (port 7860) — `app.py`  
   Chat-first HUD with tabs: Repository Map, GDrive, T.I.A., Datasets, Coding Agent,
   Code Executor, AppScript Factory, and Workflow Generator.

2. **FastAPI sidecar** (port 8000) — `main_api.py`  
   REST API consumed by the Vercel Command Deck. Handles RAG ingest/query, authenticated
   GitHub file commits, and tunnel-reachability probes.

The RAG engine (`services/rag_hub.py`) builds a FAISS IndexFlatIP over 1200-char
sliding-window chunks of Master Harvest text fragments. Embeddings are generated by
`sentence-transformers/all-MiniLM-L6-v2` (384-dim, ~90 MB, CPU-friendly).

## RAG_ENGINE_DETAILS

**Module:** `services/rag_hub.py`  
**Index type:** FAISS `IndexFlatIP` (cosine similarity via L2-normalized vectors)  
**Embedding model:** `sentence-transformers/all-MiniLM-L6-v2` — 384 dimensions  
**Chunk size:** 1200 characters, 150-character overlap  
**Default fragment globs:**
- `TIA_MASTER_HARVEST.txt`
- `AETHER_HARVEST_COMPLETE.md`
- `harvest_report.md`
- `data/master_harvest_manifest.json`
- `data/AETHER_HARVEST_DISCOVERY_MANIFEST.json`
- `data/master_harvest/INDEX.md`

**Persistence:** `data/vector_store/harvest.index` (FAISS binary) +
`data/vector_store/harvest_meta.json` (chunk metadata).

**Thread safety:** `threading.Lock()` guards all read/write operations on the index.
The module-level singleton `get_hub()` is safe for concurrent FastAPI requests.

**Lazy loading:** `faiss` and `sentence_transformers` are imported inside methods to
keep import-time fast for Streamlit cold start.

## GITHUB_COMMIT_BRIDGE

**Endpoint:** `POST /v1/system/commit`  
**Authentication:** constant-time HMAC comparison of `HF_TOKEN` from `X-HF-Token` header.  
**Action:** for each file in the request, performs a GitHub Contents API GET to retrieve
the current blob SHA (update), then PUT to create or update the file.  
**Default org:** `DJ-Goana-Coding`  
**Path validation:** blocks absolute paths, backslashes, control chars, and `..` segments.  
**Required env:** `HF_TOKEN` (auth gate), `GH_TOKEN` or `GITHUB_TOKEN` (GitHub push).

## GEMINI_KEY_ROTATION

**Module:** `services/gemini_rotator.py`  
**Pattern:** Round-robin failover across multiple Gemini API keys on `429`/`ResourceExhausted`.  
**Key discovery order:** `GEMINI_API_KEY` → `GEMINI_API_KEY_2`, `GEMINI_API_KEY_3`, … →
comma-separated `GEMINI_API_KEYS`.  
**Thread safety:** `threading.Lock()` on the cursor.  
**Public API:** `load_keys()`, `get_rotator()`, `generate_content()`, `reset_rotator()`.

## GDRIVE_INTEGRATION

**Module:** `services/gdrive_connector.py`  
**Method:** rclone with service-account JSON written to `~/.config/rclone/rclone.conf`
at startup. `GDRIVE_SERVICE_ACCOUNT_JSON` env var contains the JSON payload.  
**Operations:** list directory (`list_gdrive`), sync from GDrive (`sync_from_gdrive`),
status check (`get_gdrive_status`).  
**Targets:** defined in `GDRIVE_TARGETS` dict — maps local paths to remote GDrive paths.

## DEV_TRAPS

### Trap 1 — Cold-Start RAG Empty Index
`GET /v1/stats` returns `{"loaded": false, "chunks": 0}` on first container start because
`data/vector_store/harvest.index` does not exist. **Fix:** call `POST /v1/ingest` once
after deployment or pre-build the index and commit it.

### Trap 2 — CORS 403 from Vercel Deck
`ALLOWED_ORIGINS` in `main_api.py` is a hard-coded list. If the Vercel deployment URL
changes, browser preflights fail silently. **Fix:** keep `CITADEL_COMMAND_DECK_ORIGIN`
in sync with the live Vercel URL.

### Trap 3 — 503 on /v1/system/commit
Raised when `HF_TOKEN` or `GH_TOKEN` is absent from the HF Space secrets. The `/healthz`
probe does NOT check for these. **Fix:** check `/v1/system/tunnels` — it reports token
presence as booleans without ever logging the values.

### Trap 4 — Rate-Limit Loop on Gemini
If all Gemini keys are exhausted, `generate_content()` re-raises the last
`ResourceExhausted` error. Without backoff the caller can immediately retry, creating a
tight error loop. **Fix:** wrap `generate_content()` calls with exponential backoff or
a circuit breaker.

### Trap 5 — VAMGUARD / VANGUARD Naming Mismatch
Many documentation files, template directories (`vamguard_templates/`), and workflow
YAMLs reference "VAMGUARD". Downstream code that reads the exact string "VANGUARD" will
fail to match. Treat these as semantic aliases; do not rename without updating all
references.

### Trap 6 — rclone Not Installed
`services/gdrive_connector.py` shells out to `rclone`. The HF Space container needs
`rclone` listed in `packages.txt` or the GDrive tab will always show "rclone not found".

### Trap 7 — Streamlit / FastAPI Port Conflict
Both processes must be started by `scripts/start_hub.sh`. If only Streamlit starts, the
entire REST API is dark. The HF Space `CMD` must invoke `start_hub.sh`, not `streamlit run`.

### Trap 8 — GitHub Path Traversal Guard
`POST /v1/system/commit` validates every file path against a strict allow-list. Paths with
`..`, absolute paths, backslashes, or control chars raise HTTP 400. Build file paths as
clean relative POSIX strings with no leading slash.
