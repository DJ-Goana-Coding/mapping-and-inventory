## 🚀 **DEPLOYMENT QUICK START**

### **Option 1: HuggingFace Space (Recommended)**

```bash
# 1. Create Space at https://huggingface.co/new-space
# 2. Clone Space repo
git clone https://huggingface.co/spaces/YourOrg/TIA-ARCHITECT-CORE

# 3. Copy files
cp TIA_UNIFIED_APP.py TIA-ARCHITECT-CORE/app.py
cp requirements.txt TIA-ARCHITECT-CORE/
cp README.md TIA-ARCHITECT-CORE/

# 4. Set secrets (in HF Space Settings)
GEMINI_API_KEY=your-key
HF_TOKEN=your-token

# 5. Push & deploy
cd TIA-ARCHITECT-CORE
git add . && git commit -m "Deploy TIA Master Build" && git push
```

### **Option 2: Local Development**

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Set environment variables
export GEMINI_API_KEY=your-key
export HF_TOKEN=your-token

# 3. Run
streamlit run TIA_UNIFIED_APP.py
```
